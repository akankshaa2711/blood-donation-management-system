This directory contains images used in the get-started page of the blood donation application.

## How to Add Images

### Option 1: Use Real Images
1. Find or create appropriate blood donation related images
2. Ensure images are properly licensed for your use
3. Optimize images for web (compress, resize appropriately)
4. Save images in JPG, PNG, or WebP format
5. Place images in this directory with the exact filenames listed above

### Option 2: Generate Placeholder Images
1. Open the `placeholder-images.html` file in a web browser
2. This will show you what the placeholder images look like
3. Use the `generate-placeholders.js` script in your browser console:
   - Open any webpage in your browser
   - Open Developer Tools (F12)
   - Go to Console tab
   - Copy and paste the content of `generate-placeholders.js`
   - Run `generateAllPlaceholders()` to generate all images
   - Download each image and save to this directory

### Option 3: Use Stock Photos
Consider using stock photo services like:
- Unsplash (free)
- Pexels (free)
- Pixabay (free)
- Shutterstock (paid)
- iStock (paid)

Search for terms like:
- "blood donation"
- "medical donation"
- "healthcare volunteer"
- "hospital blood"
- "medical professional"

## Image Guidelines

- **Format**: JPG, PNG, or WebP
- **Size**: Optimize for web (under 500KB per image)
- **Dimensions**: Use the recommended sizes above
- **Quality**: High quality but web-optimized
- **Content**: Relevant to blood donation and healthcare
- **Licensing**: Ensure proper usage rights

## Current Status

The get-started page is currently set up to display these images. If images are missing, the page will show broken image icons. Make sure all referenced images are present in this directory.

## Testing

After adding images:
1. Refresh the get-started page in your browser
2. Verify all images are displaying correctly
3. Check that images are responsive on different screen sizes
4. Ensure image loading doesn't significantly impact page performance
