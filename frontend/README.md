# Blood Donation Management System

A beautiful and modern Ionic Angular app for managing blood donation activities, connecting donors with healthcare facilities, and providing comprehensive information about blood donation.

## 🎯 Features

### 🏠 Home Page
- **Stunning Design**: Beautiful hero section matching your template design
- **Film Reel Aesthetic**: Unique left panel with film reel design elements
- **Image Collage**: Right panel with placeholder images for donor activities
- **Call-to-Action**: Prominent "Get Started" button to guide users
- **Feature Highlights**: Information about why blood donation matters

### 🔐 Login Page
- **Modern UI**: Clean, gradient-based design with smooth animations
- **Form Validation**: Email and password validation
- **Guest Access**: Option to continue as guest
- **Responsive Design**: Works perfectly on all device sizes

### 👥 Donors Page
- **Search & Filter**: Find donors by name or blood group
- **Statistics Cards**: Overview of total donors, available donors, and recent donations
- **Donor Cards**: Detailed information including:
  - Name and location
  - Blood group
  - Contact information (phone, email)
  - Last donation date
  - Availability status
- **Interactive Elements**: Click to view details, contact donors

### 👤 Profile Page
- **User Information**: Personal and medical details
- **Donation History**: Recent donations with dates and locations
- **Quick Stats**: Total donations and lives saved
- **Settings Access**: Password change, notifications, privacy
- **Avatar Management**: Profile picture with edit functionality

### ℹ️ About Page
- **Mission Statement**: Clear explanation of the app's purpose
- **How It Works**: Step-by-step donation process
- **Blood Type Information**: Comprehensive compatibility guide
- **Safety Requirements**: Donation eligibility criteria
- **Contact Information**: Multiple ways to get support

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn
- Ionic CLI

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd bloodDonationApp
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   ionic serve
   ```

4. **Open your browser**
   Navigate to `http://localhost:8100`

### Building for Production

```bash
# Build the app
ionic build

# Build for Android
ionic capacitor build android

# Build for iOS
ionic capacitor build ios
```

## 📋 Blood Type Information

The app includes comprehensive information about all 8 blood types:
- **A+**: 30% of population
- **A-**: 6% of population
- **B+**: 9% of population
- **B-**: 2% of population
- **AB+**: 4% of population
- **AB-**: 1% of population
- **O+**: 39% of population
- **O-**: 9% of population

Each blood type includes compatibility information for donations and receiving.
